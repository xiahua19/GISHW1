1.      属性表操作

1.1.      查看属性表

您可以通过“右键图层列表中的某个图层->打开属性表”来查看对应图层的属性表，系统将以弹窗的形式为您展示属性表。

1.2.      通过属性表选择要素

在图层属性表窗口中，您可以点击属性表右侧空白单元格来选择要素，被选择的要素同时会在图层和显示中得到更新。

1.3.      增加字段

在图层属性表窗口中，您可以点击“选项->添加字段”来为当前显示在属性表窗口中的图层添加字段，设置好相应的参数之后，您就可以成功添加字段。

1.4.      删除字段

您只需要在图层属性表窗口中右键属性表上侧表头单元格，点击“删除字段”，即可删除对应图层的对应字段。

1.5.      属性查询

您可以通过“右键图层列表中的某个图层->按属性选择”来打开属性查询对话框，对话框右侧列出了该图层所有的字段，双击某个字段可以在查询语句中添加字段名称，在查询语句输入框中输入对应的查询语句，然后点击应用或确定，便可以执行查询。如果查询语句无效，则会弹出对话框提示。

2.      编辑数据操作

要想进行数据编辑，首先要进入编辑模式，您可以通过点击工具条中“图形编辑器”按钮来开启/关闭编辑状态，开启时，必须选择一个可见图层操作。此外，所有在系统中编辑修改的数据均为内存操作，如果您需要保存修改，请右键图层列表中的图层，点击“另存为图层”，选择路径（也可以选择覆盖原始图层文件）即可保存修改，这样的修改和存储策略操作类似于Microsoft Word等文本编辑器软件。

2.1.      编辑属性数据

在编辑模式下，打开【任何】图层属性表，双击属性表中的单元格，可以对数据进行修改。如果提示出错，请检查输入的数据类型是否匹配。

2.2.      编辑图形数据

在编辑模式下，您可以对图形数据进行移动、编辑和创建操作。

1、移动要素

在编辑模式下，首先点击工具条中的“选择”按钮（也即“箭头”），对正在编辑的图层进行移动要素的选择，选择完毕后，您可以点击工具条中的“移动要素”按钮，进入移动模式，移动模式下鼠标按下并拖动可以对正在编辑图层中所选中的要素进行拖动，鼠标松开则完成移动。

2、创建要素

在编辑模式下，点击工具条中的“创建要素”按钮，进入创建要素模式，此时鼠标会变成十字丝，顺次点击地图中位置可以进行新要素的内容输入，在地图任意位置右键可以选择“结束部分”或“结束描绘”，前者表示完成要素的part绘制，后者表示完成整个要素的绘制。绘制完毕要素后，您可以在属性表中修改要素的各个属性。

3、编辑结点

在编辑模式下，首先使用“选择要素”工具选择且只选择一个要素，然后点击工具条中的“编辑结点”按钮，进入编辑要素模式，此时被选择的要素会显示为可编辑结点模式，您可以在任意结点上按下鼠标，拖动鼠标，松开鼠标，可以完成对结点的修改。
